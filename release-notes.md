# GraphXR Viewer v3.7.0

A standalone viewer application for GraphXR project files (.graphxr).

## What's Included

### Windows
- **GraphXR Viewer Setup 3.7.0.exe** - Windows installer (90 MB)
- **GraphXR Viewer-3.7.0-win.zip** - Portable Windows version (119 MB)

### macOS
- **GraphXR Viewer-3.7.0.dmg** - macOS installer for Intel Macs (116 MB)
- **GraphXR Viewer-3.7.0-arm64.dmg** - macOS installer for Apple Silicon (111 MB)
- **GraphXR Viewer-3.7.0-mac.zip** - Portable macOS version for Intel (112 MB)
- **GraphXR Viewer-3.7.0-arm64-mac.zip** - Portable macOS version for Apple Silicon (107 MB)

### Linux
- **GraphXR Viewer-3.7.0-arm64.AppImage** - Linux AppImage (120 MB)
- **graphxr-viewer_3.7.0_arm64.deb** - Debian package (81 MB)

## Features

- 3D Graph Visualization with interactive navigation
- Information panels for nodes and relationships
- Visual legends for graph elements
- File association for .graphxr files
- Cross-platform support
- Clickable Kineviz logo linking to kineviz.com
